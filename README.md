# Data-Preparation-and-First-Supervised-Model
Applied Artificial Intelligence &amp; Machine Learning.

## Assignment 1: Data Preparation and First Supervised Model
This assignment introduces you to the full modeling workflow used throughout this course. You will prepare a real dataset for modeling, fit a supervised learning model, and explain your decisions clearly and defensibly.
Learning Objectives
- Understand how data preparation choices affect models
- Practice fitting and interpreting a regression model
- Develop the ability to explain modeling decisions in plain language
- Separate technical execution from analytical judgment
Dataset
You will use the provided dataset: insurance.csv.

Each row represents an individual insurance customer, with demographic and behavioral attributes and a target variable representing medical charges.

### Part 1: Data Understanding and Preparation
In a Google Colab notebook:
- Load the dataset
- Clearly state what one row represents
- Identify the target variable
- Inspect data types, missing values, and anomalies
- Prepare the data appropriately for regression (encoding, transformations, etc.)

You must justify each major preparation decision in words.

### Part 2: Supervised Model (Regression)
Fit an Ordinary Least Squares (OLS) regression model using the prepared data.

You must:
- Report the estimated coefficients
- Report R-squared
- Interpret at least two coefficients in plain language
- Explain what the model is claiming about the relationship between inputs and the target
You must justify each major preparation decision in words.

### Part 3: Reflection and Limitations
Answer the following in your notebook or reasoning sheet:
- One model assumption you are confident about
- One model assumption you are uncertain about
- One reason you might distrust the model’s results
You must justify each major preparation decision in words.
